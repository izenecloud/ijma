Instruction for Manual Check (Please read it carefully in the first time)

For Each sentence below, there are two approaches to do the manual check.

The first approach is Recommended by single click (no double click) the wrong segmented word directly. If a word is wrong, its background color is red, like 体罰 and if it is correct, its background is white, like 体罰. If you don't think a wrong word (with red background color) is wrong, just click that word one more time. Here you can practise it by click the word: 体罰.

The second approach is Not Recommended as it is much easier to make mistakes. This approach is to input Suggestion (i.e., the correct segmentation) directly. It include three steps:

  1. Click "Copy Original Text" button to get the original text;
  2. Separate the words with spaces (Avoid to enter other characters and Make sure you are using English Input Method);
  3. Click "Apply" button to apply your input.

At the beginning of Apply Suggestion(step 3), the program would check whether the suggestion contains invalid characters (compared with the original text, only allow to add English space characters). So if you choose this approach, please Pay Attention on step 2.

If input the invalid character, for example, the original text is 教師を足で蹴(け)って, and user's input is "教師 を足で;蹴(け)って". We can see there is a non-space character ";" behind "で". The error message would be "Invalid User Input(Non-Space) ';' at index 6 ( を足で;蹴(け)". this message contains that invalid character (;), its index (6) and the surrounding string ( を足で;蹴(け)(in the last parenthesis).

It is Not Recommended to use two approaches for a sentence at the same.

All the modifications would be saved automatically (the checking interval is 3 minutes); Or you can click the button "Save Changes"(at the end of each sentence block) to save all the modifications. "Save Changes" button is enable if there are some modifications after be saved last time.
